# Utils 工具函数列表

| 工具函数名称 | 描述 |
| --- | --- |
| `formatTokenAmount` | 代币数量格式化工具，用于将最小单位（如 lamports）转换为可读的代币数量，并支持千分位格式化 |
| `solToLamports` | 将用户输入的 SOL 金额转换为 lamports（乘以10^9） |
| `formatValue` | 格式化数值显示为紧凑的美元格式（K、M、B） |
| `formatTokenAmountWithDecimals` | 根据指定的代币精度格式化代币数量，支持智能小数位数调整 |
| `formatSmallNumber` | 格式化小数字，使用下标数字表示前导零数量 |
| `cluPrice` | 计算代币的美元价值 |
| `formatPercentage` | 格式化百分比，自动添加符号和颜色 |
| `formatAmountInYiOrWan` | 格式化金额为亿/万单位 |
