# cluPrice

计算代币的美元价值。

## 代码演示

import CluPriceDemo from './components/CluPrice'

<CluPriceDemo />

## API

### 参数

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 代币数量（最小单位） | `string \| number \| undefined \| null` | - |
| tokenDecimals | 代币精度（如9、6等） | `number` | - |
| price | 单个代币的美元价格 | `string \| number \| undefined \| null` | - |

### 返回值

返回格式化后的美元价值字符串（带$符号和K/M/B后缀）。

### 特性

- **高精度计算**: 使用 BigNumber 确保计算精度
- **自动格式化**: 大额金额自动使用K、M、B后缀
- **多精度支持**: 支持不同精度的代币（SOL 9位、SPL Token 6位等）
- **错误处理**: 对无效输入返回$0

## 使用示例

### 基本用法

```tsx
import { cluPrice } from '@miaoma-rhooks/utils'

// 计算 SOL 的美元价值（9位精度）
cluPrice('1000000000', 9, 100)  // '$100.00'  (1 SOL * $100)
cluPrice('1500000000', 9, 100)  // '$150.00'  (1.5 SOL * $100)

// 计算 SPL Token 的美元价值（6位精度）
cluPrice('1000000', 6, 10)      // '$10.00'   (1 Token * $10)
cluPrice('1500000', 6, 10)      // '$15.00'   (1.5 Token * $10)
```

### 不同价格

```tsx
cluPrice('1000000000', 9, 50)   // '$50.00'   (1 SOL * $50)
cluPrice('1000000000', 9, 200)  // '$200.00'  (1 SOL * $200)
```

### 小数数量

```tsx
cluPrice('100000000', 9, 100)   // '$10.00'   (0.1 SOL * $100)
cluPrice('10000000', 9, 100)    // '$1.00'    (0.01 SOL * $100)
```

### 大额数值

```tsx
cluPrice('1000000000000', 9, 100)   // '$100.00K'  (1000 SOL * $100)
cluPrice('10000000000000', 9, 100)  // '$1.00M'    (10000 SOL * $100)
```
